V0.16a

Welcome to the 16th release of OvBB, the open source discussion board system that aims to mimic vBulletin 2.x.x in both look and behavior while incorporating new features and functionality. This release fixes a handful of bugs, changes some behavior, and adds some new features, however the most important difference is the addition of PostgreSQL support and the ability to easily add support for more databases in later versions. As always, check out changelog.html for details on exactly what has changed since V0.15a.

Requirements

Although OvBB works on most setups, you should make sure your environment fulfills these requirements:

  • About 1.5 megabytes of free storage space
  • PHP 4.3.0 or better
  • Access to a database (or the privileges to create one) on a MySQL or PostgreSQL server

Additionally, if you want to use the Image Verification (CAPTCHA) feature in the registration of new users, you'll need to have the GD Graphics Library and the FreeType Library installed and configured in your environment.

Fresh Installation

  1. Upload the entire contents of the upload directory to a location of your choice on your server.
  2. Begin the installation by accessing the setup subdirectory via your Web browser.
  3. Select "Fresh Installation" as the type, simply follow the instructions, and post any problems you run into on the OvBB Project Support Forum.
  4. Be sure to delete the setup subdirectory immediately after installation is completed.

Upgrade From V0.15a

  1. Upload the entire contents of the upload directory to the location of your forums, overwriting files as necessary.
  2. Begin the upgrade by accessing the setup subdirectory via your Web browser.
  3. Select "Upgrade From V0.15a" as the type, simply follow the instructions, and post any problems you run into on the OvBB Project Support Forum.
  4. Be sure to delete the setup subdirectory immediately after the upgrade is completed.
